    Ontwerpgeschiedenis Boekje Kaft

    Voor het eindcijfer van ontwerpgeschiedenis moest ik een boekje maken met daarin alle stof van de lessen en ook nog extra informatie waarvan ik zelf vond die hierbij aansloot. Dit boekje kan ik niet laten zien omdat we dit moesten inleveren en dit hebben we niet teruggekregen. Wel kan ik de kaft laten zien. De kaft heb ik ontworpen in de stijl van het Bauhaus. Over het Bauhaus heb ik veel geleerd bij ontwerpgeschiedenis. Dit vind ik een speciaal ontwerp van mijn omdat het heel erg anders is dan de rest, bij veel vakken val ik in herhaling qua ontwerp, maar dit is voor mij een uniek ontwerp.

    New Product Development Posters

    Bij New Product Development moest ik in een team van drie personen een concept bedenken. Dit vak was heel veel werk, misschien wel het meeste werk van de hele opleiding. Het was niet echt een vak dat mij interesseerde, maar ik heb er wel heel veel van geleerd. Deze posters zijn het resultaat hiervan. Ik ben trots op deze posters omdat ze heel veel werk kostten en omdat ik veel nieuwe dingen heb toegepast bij de hele opdracht van New Product Development.

    Prototyping Productvideo

    Voor prototyping moest ik in een duo veel verschillende ideeën testen op mensen op straat in Amsterdam. Ook moest ik een filmpje maken over het gebruik van één van onze ideeën. Met de hand van een storyboard hebben we dit filmpje gemaakt. In mijn verleden heb ik veel filmpjes gemaakt op straat en thuis. Hiervan wist ik dus al iets meer af en heb ik toegepast op iets nieuws en ik heb er ook weer nieuwe dingen van geleerd, zoals het gebruik van een storyboard.

    Eindopdracht Inleiding Programmeren

    Het vak programmeren was samen met internetstandaarden het vak waar ik het meeste naar vooruit keek, van deze vakken wilde ik heel veel leren. Bij dit vak heb ik heel veel geleerd dat ik wilde leren. In de eindopdracht heb ik alle kennis van dit vak toegepast en een keyboard drum machine gemaakt. Dit is één van de opdracht waarbij ik echt lol had om het te maken en ik het gevoel had dat ik heel veel nuttigs geleerd heb.

    Project 2a Eindproduct

    Het tweede project was een teamproject, ik zat in een team van vier. Dit was het eerste project in teamverband waar verschillende teamleden andere take op zich namen. Ik was samen met een ander teamlid bezig met de vormgeving en HTML, CSS en Javascript van de website en de andere teamleden met de content. Ik ben trots op dit eindproduct omdat bij dit project goed de verschillende kwaliteiten van de verschillende teamleden zijn toegepast.
